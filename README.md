# 📸 Fotogram

Eine interaktive Fotogalerie, entwickelt mit HTML, CSS und JavaScript.

## 🌐 Live Demo

👉 https://janetbughardt.github.io/fotogram/

## 📖 Projektbeschreibung

Fotogram ist eine interaktive Bildergalerie mit Fokus auf Benutzerfreundlichkeit und responsivem Design. Bilder können in einer vergrößerten Ansicht betrachtet und komfortabel durchgeblättert werden.

## ✨ Funktionen

- Bildergalerie
- Overlay mit Großansicht
- Vor- und Zurück-Navigation
- Responsive Design
- Dynamische Darstellung der Bilder

## 🛠️ Verwendete Technologien

- HTML5
- CSS3
- JavaScript (ES6)
